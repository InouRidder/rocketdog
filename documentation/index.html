About

Muziq is a clean and professional site template, perfect for Music Event, Live music and ticket music web sites. Built on the Advanced Bootstrap 3 framework with HTML5 / CSS3. Its responsive too - will fit great on all devices. More features with , and much more gives you more space to work with and also with a few lines of code.

Clean coded and Easy updation are the key features. Go ahead to know more.

CSS files and structure

I have used Eleven CSS files in this theme.

  1. Muziq.css - Contains all of the specific stylings for the page.
  2. bootstrap.min.css - Bootstrap css.
  3. font-awesome.min.css - Fontawesome Icons.
  4. owl.carousel.css - Contains all of the specific stylings for Owl Carousel
  5. owl.theme.css - Contains all of the specific stylings for Owl Carousel
  6. owl.transitions.css - Contains all of the specific stylings for Owl Carousel
  7. YTPlayer.css - Contains all of the specific stylings for Youtube plugin
  8. swipebox.css - Contains all of the specific stylings for Swipebox plugin
Style.css file contains all of the specific stylings for the page. The file is separated into sections using:

Javascript

This theme imports Twenty Javascript files.

  1. modernizr.custom.js - Modernizer plugin.
  2. jquery.js - To Enable jQuery.
  3. classie.js - Class Helper plugin.
  4. pathLoader.js - Preloading plugin.
  5. owl.carousel.min.js - Carousel plugin.
  6. jquery.inview.js - Inview jquery plugin.
  7. jquery.nav.js - Navigation plugin.
  8. jquery.mb.YTPlayer.js - Youtube plugin.
  9. jquery.form.js - Form plugin.
  10. jquery.validate.js - Validation plugin.
  11. bootstrap.min.js - Plugin for Bootstrap framework.
  12. default.js - Template custom plugin.
  13. plugins.js - Template second custom plugin.
  14. jquery.isotope.min.js - Isotope Plugin.
  15. jquery.prettyPhoto.js - Pretty Photo Plugin.
  16. jquery.swipebox.js - Lightbox Plugin.

    Change color
    1. Please copy the color you want and add it before tags.
    2. Then add class in body tags
    3. Yellow class: paletteyellow
    4. Blue class: palettelightblue
    5. Purple class: palettepurple
    6. Green class: palettegreen
    7. Militar class: palettemilitar
